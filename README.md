# 🍽️ FoodieHub – Restaurant Listing Website

FoodieHub is a React-based web application that displays a list of restaurants, allows users to search and filter through them, and view detailed menus for each restaurant.  
It is inspired by Swiggy and other online food delivery platforms.

---

## 🚀 Features

- 📋 **Restaurant List** – Browse a variety of restaurants with images, names, ratings, and cuisines.
- 🔍 **Search Functionality** – Search restaurants by name.
- 🧮 **Filter Options** – Filter restaurants based on ratings or other criteria.
- 📜 **Restaurant Menu** – View detailed menus with food items and prices.
- ⚡ **Shimmer UI** – Loading animation for a better user experience.
- 📱 **Responsive Design** – Works on desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, JavaScript, HTML, CSS
- **Styling:** Custom CSS (No Tailwind)
- **State Management:** React Hooks (`useState`, `useEffect`)
- **API:** Swiggy public API (for mock data)
- **Build Tool:** Create React App (CRA)
